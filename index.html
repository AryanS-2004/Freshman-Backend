Data Structures and Algorithms

The most important thing to do if you want to work in big MNCs for SDE or any other software development roles is DSA. A data structure is a data organization, management, and storage format that is usually chosen for efficient access to data.

Development

Software development involves writing and maintaining the source code, but in a broader sense, it includes all processes from the conception of the desired software through to the final manifestation of the software, typically in a planned and structured process.

Competitive Programming

Competitive programming is a programming competition generally involving the host presenting a set of logical or mathematical problems to the contestants and contestants are required to write computer programs capable of solving each problem.
